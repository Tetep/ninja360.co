7 · Statement of Work (SOW)

Objective

Automate the front of the proven Hasan system so a finished shoot becomes a review-ready Shedit draft with near-zero manual handling — while capturing the show from the same visit.

Success metrics
  • Shoot-to-publish time — baseline → target (e.g. 24h)
  • Weeks shipped ÷ 13 — sprint delivery rate
  • % drafts passing review in ≤1 pass
Scope (five workstreams)
  • 1 · Shoot SOP — standardized 2-track capture so automation can run
  • 2 · Warp Speed — auto-sort → auto-color → auto-nadir → Shedit into 03_EXPORTS
  • 3 · Templates & assets — Ninja LUT · Nadir preset · story templates · tag taxonomy
  • 4 · Automation — extend the existing Apps Script (Pavan)
  • 5 · Show layer — on-location + studio segments feed "Found in KC"

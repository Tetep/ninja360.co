02 · How the Money Splits — Read This First

The split is new, and it comes down to two ledgers that never touch: Ledger A — your existing Ninja clients (yours alone) and Ledger B — the new Ninja Dojo engine you build with the crew. This is the rule everything else obeys, so it goes up front.

Two Ledgers, Never Touching
Ledger A · Existing Ninja Clients
Your existing book — LiveSmart, Kobler, 54th Street Grill Chain (32 units), Immersive Data Solutions (IDS + its retail subcontracts) — deals Tim already closed. Tim's alone. Funds Tim and the business. Never enters partner economics. When Gabe or Pavan help on one of these shoots, they're paid the flat field rate — set in Partner Pay, kept fully separate from engine pay.
Ledger B · The New Ninja Dojo Engine
The new engine you build together — Tim × Gabe × Pavan. Starts at $0. The only thing the crew shares in. Funded only by fresh clients + the show. No fresh revenue → no engine payout. The LLC owns all IP; partners earn from what they build, they don't own the company.
To lock with the crew: the per-leg numbers below (rates, per-episode, splits, rev-share). They're marked — fill them in the room, in writing, before work starts.

05 · How It's Run — The Monday Loop

Defined Monday. Tracked all week. Followed up Friday. One simple loop, run on the dojo.

WhenWhat happens
Monday · Dojo30 minutes. Set the week's one objective. Fill the Mission Board: each card = owner · deliverable · due date. Pull each card from the leg it serves.
All week · ThreadUpdates posted to the thread. No meetings. The board is the source of truth.
Friday · WrapEveryone posts their wins. Count the stars. What didn't ship rolls to Monday with a reason.
The Floor is the only surface you touch. Set the board Monday, post to the thread all week, count stars Friday — the rest of the dojo holds itself. You run the rhythm; the rhythm runs the plan.
